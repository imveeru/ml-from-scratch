# Decision Tree Regressor Example

This example demonstrates the `RegressionTree` on temperature data.

## Description
-   **Dataset**: `TempLinkoping2016.txt`.
-   **Task**: Regression.
-   **Model**: Regression Tree.
-   **Visualization**: Plots the regression line over the data points.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.decision_tree_regressor
```

## Output
-   Prints Mean Squared Error.
-   Displays a plot showing the regression model's predictions.
