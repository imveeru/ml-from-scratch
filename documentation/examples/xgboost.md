# XGBoost Example

This example demonstrates XGBoost for classification.

## Description
-   **Dataset**: Sklearn `iris`.
-   **Task**: Multi-class Classification.
-   **Model**: XGBoost (Extreme Gradient Boosting).
-   **Visualization**: Plots classification results.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.xgboost
```

## Output
-   Prints Accuracy.
-   Displays classification plot.
