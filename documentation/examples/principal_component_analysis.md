# Principal Component Analysis (PCA) Example

This example demonstrates dimensionality reduction using PCA.

## Description
-   **Dataset**: Sklearn `digits` (64 dimensions).
-   **Task**: Dimensionality Reduction.
-   **Method**: PCA (Reduce to 2 Principal Components).
-   **Visualization**: Plots the data in the new 2D latent space, colored by digit label.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.principal_component_analysis
```

## Output
-   Displays a scatter plot showing the separation of digits in 2D space.
