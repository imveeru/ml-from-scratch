# Partitioning Around Medoids (PAM) Example

This example demonstrates K-Medoids clustering.

## Description
-   **Dataset**: Sklearn `make_blobs`.
-   **Task**: Clustering.
-   **Model**: PAM (K-Medoids, k=3).
-   **Visualization**: Plots Predicted vs Actual Clusters.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.partitioning_around_medoids
```

## Output
-   Displays two plots: PAM Clustering vs Actual Clustering.
