# Linear Discriminant Analysis (LDA) Example

This example demonstrates LDA for classification.

## Description
-   **Dataset**: Sklearn `iris` (filtered to 2 classes).
-   **Task**: Binary Classification.
-   **Model**: LDA.
-   **Visualization**: Plots classification results.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.linear_discriminant_analysis
```

## Output
-   Prints Accuracy.
-   Displays classification plot.
