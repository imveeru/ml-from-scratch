# FP-Growth Example

This example demonstrates the FP-Growth algorithm for frequent pattern mining.

## Description
-   **Dataset**: Hardcoded dummy transaction dictionary.
-   **Task**: Find frequent itemsets.
-   **Parameters**:
    -   `min_sup`: 3

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.fp_growth
```

## Output
-   Prints the input transactions.
-   Prints found Frequent Itemsets.
