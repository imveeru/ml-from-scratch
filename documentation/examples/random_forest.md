# Random Forest Example

This example demonstrates Random Forest Classification.

## Description
-   **Dataset**: Sklearn `digits`.
-   **Task**: Multi-class Classification.
-   **Model**: Random Forest (100 Decision Trees).
-   **Visualization**: Plots classification results.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.random_forest
```

## Output
-   Prints Accuracy.
-   Displays classification plot.
