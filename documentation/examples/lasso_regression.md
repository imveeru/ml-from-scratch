# Lasso Regression Example

This example demonstrates Lasso ($L_1$) Regression.

## Description
-   **Dataset**: `TempLinkoping2016.txt`.
-   **Task**: Regression.
-   **Model**: Lasso Regression.
-   **Features**: Polynomial degree 13.
-   **Visualization**: Plots Error vs Iterations and Final Prediction.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.lasso_regression
```

## Output
-   Displays Error Plot.
-   Prints MSE.
-   Displays Regression Plot.
