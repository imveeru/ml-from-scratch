# Multi-Class LDA Example

This example demonstrates Multi-Class Linear Discriminant Analysis for dimensionality reduction.

## Description
-   **Dataset**: Sklearn `iris` (3 classes).
-   **Task**: Dimensionality Reduction.
-   **Model**: Multi-Class LDA.
-   **Visualization**: Projects 4D Iris data onto 2D space and plots it.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.multi_class_lda
```

## Output
-   Displays a scatter plot of the data projected onto the top 2 discriminants.
