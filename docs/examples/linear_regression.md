# Linear Regression Example

This example demonstrates basic Linear Regression.

## Description
-   **Dataset**: Sklearn `make_regression` (Synthetic).
-   **Task**: Regression.
-   **Model**: Linear Regression.
-   **Visualization**: Plots Error vs Iterations and Final Regression Line.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.linear_regression
```

## Output
-   Displays Error Plot.
-   Prints MSE.
-   Displays Regression Plot.
