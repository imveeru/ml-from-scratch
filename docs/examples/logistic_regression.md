# Logistic Regression Example

This example demonstrates Logistic Regression for binary classification.

## Description
-   **Dataset**: Sklearn `iris` (Filtered to 2 classes).
-   **Task**: Binary Classification.
-   **Model**: Logistic Regression (Gradient Descent).
-   **Visualization**: Plots classification results.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.logistic_regression
```

## Output
-   Prints Accuracy.
-   Displays classification plot.
