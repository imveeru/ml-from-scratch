# Gradient Boosting Classifier Example

This example demonstrates Gradient Boosting for classification.

## Description
-   **Dataset**: Sklearn `iris`.
-   **Task**: Multi-class Classification.
-   **Model**: Gradient Boosting Classifier.
-   **Visualization**: Plots classification results in 2D.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.gradient_boosting_classifier
```

## Output
-   Prints Accuracy.
-   Displays a scatter plot of the results.
