# K-Means Clustering Example

This example demonstrates K-Means Clustering.

## Description
-   **Dataset**: Sklearn `make_blobs`.
-   **Task**: Clustering.
-   **Model**: K-Means ($k=3$).
-   **Visualization**: Plots Predicted vs Actual Clusters in 2D.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.k_means
```

## Output
-   Displays two plots: K-Means Clustering vs Actual Clustering.
