# Decision Tree Classifier Example

This example demonstrates the `ClassificationTree` on the Iris dataset.

## Description
-   **Dataset**: Sklearn `iris`.
-   **Task**: Multi-class Classification (3 flower species).
-   **Model**: Classification Tree.
-   **Visualization**: Plots the decision regions in 2D (using PCA).

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.decision_tree_classifier
```

## Output
-   Prints Accuracy.
-   Displays a 2D plot of the classification results.
