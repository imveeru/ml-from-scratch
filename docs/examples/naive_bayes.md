# Naive Bayes Example

This example demonstrates Gaussian Naive Bayes classification.

## Description
-   **Dataset**: Sklearn `digits`.
-   **Task**: Multi-class Classification.
-   **Model**: Naive Bayes.
-   **Visualization**: Plots classification results.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.naive_bayes
```

## Output
-   Prints Accuracy.
-   Displays classification plot.
