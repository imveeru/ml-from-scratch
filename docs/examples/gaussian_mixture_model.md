# Gaussian Mixture Model (GMM) Example

This example demonstrates GMM clustering.

## Description
-   **Dataset**: Sklearn `make_blobs` (Synthetic data).
-   **Task**: Clustering.
-   **Model**: Gaussian Mixture Model (k=3).
-   **Visualization**: Plots Predicted Clusters vs Actual Clusters.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.gaussian_mixture_model
```

## Output
-   Displays two plots: GMM Clustering vs Actual Clustering.
