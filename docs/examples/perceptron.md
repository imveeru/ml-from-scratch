# Perceptron Example

This example demonstrates the Perceptron algorithm.

## Description
-   **Dataset**: Sklearn `digits`.
-   **Task**: Multi-class Classification.
-   **Model**: Perceptron (Single Layer Neural Network).
-   **Optimization**: Gradient Descent, CrossEntropy Loss, Sigmoid Activation.
-   **Visualization**: Plots classification results.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.perceptron
```

## Output
-   Prints Accuracy.
-   Displays classification plot.
