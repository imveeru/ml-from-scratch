# Gradient Boosting Regressor Example

This example demonstrates Gradient Boosting for regression.

## Description
-   **Dataset**: `TempLinkoping2016.txt`.
-   **Task**: Regression.
-   **Model**: Gradient Boosting Regressor.
-   **Visualization**: Plots regression predictions vs actual data.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.gradient_boosting_regressor
```

## Output
-   Prints Mean Squared Error.
-   Displays a plot of Training Data, Test Data, and Predictions.
