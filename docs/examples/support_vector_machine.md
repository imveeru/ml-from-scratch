# Support Vector Machine (SVM) Example

This example demonstrates SVM Classification with a Polynomial Kernel.

## Description
-   **Dataset**: Sklearn `iris` (Filtered to 2 classes).
-   **Task**: Binary Classification.
-   **Model**: SVM (Polynomial Kernel: degree=4).
-   **Visualization**: Plots classification results.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.support_vector_machine
```

## Output
-   Prints Accuracy.
-   Displays classification plot.
