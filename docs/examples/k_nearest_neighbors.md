# K-Nearest Neighbors (KNN) Example

This example demonstrates KNN Classification on the Iris dataset.

## Description
-   **Dataset**: Sklearn `iris`.
-   **Task**: Multi-class Classification.
-   **Model**: K-Nearest Neighbors ($k=5$).
-   **Visualization**: Plots classification results in 2D.

## Usage
Run the script from the root directory:
```bash
python -m mlfromscratch.examples.k_nearest_neighbors
```

## Output
-   Prints Accuracy.
-   Displays a scatter plot of the results.
